## Quick Reference

### Tech Stack Summary
- **Frontend:** React 19.2 with Vite 7.2
- **Styling:** TailwindCSS 4.1 with custom CSS variables
- **Icons:** Lucide React 0.562
- **Audio:** Web Audio API (no external files)
- **PWA:** manifest.json + sw.js for offline support

### Key Entry Points
- **App Entry:** `src/main.jsx`
- **Root Component:** `src/App.jsx`
- **Data Store:** `src/data/workoutDatabase.js`

### Architecture Pattern
Component-based SPA with local React state management. No external backend - all workout data is embedded.

## Project Data Summary

### Exercise Library
**22 unique exercises** with:
- Display name
- Instruction text
- PNG illustration (~150KB each)

**Categories:** Warmup, Strength, Stability, Power, Finisher

### Workout Templates
**6 workout templates:**
- `gym_monday_p1` / `gym_monday_p2` - Foundation/Explosive Power
- `gym_wednesday_p1` / `gym_wednesday_p2` - Stability/Speed & Endurance
- `home_thursday_p1` / `home_thursday_p2` - Proprioception/Ski Endurance

### Program Schedule
**6-week program:**
- Weeks 1-3: Phase 1 (Foundation)
- Weeks 4-6: Phase 2 (Performance)
- Frequency: 3 days/week (Mon, Wed, Thu)

## For AI Agents

When working on this project:

1. **Data changes** ‚Üí Edit `src/data/workoutDatabase.js`
2. **UI changes** ‚Üí Edit components in `src/components/`
3. **Styling** ‚Üí Use TailwindCSS classes or edit `src/index.css`
4. **New exercises** ‚Üí Add image to `src/assets/exercises/` and update database
5. **Utilities** ‚Üí `src/utils/linearizer.js` (workout steps) or `src/utils/audioManager.js` (sounds)

### Key Patterns
- State is managed locally in components (no Redux/Context)
- Audio requires user interaction to initialize (Web Audio API requirement)
- PWA features rely on `public/manifest.json` and `public/sw.js`
